# Fast Food

A fully responsive multi-page website for a fast food restaurant called Fast Food.
Built using HTML, CSS, Bootstrap and JavaScript.

## Pages
- index.html - Home page
- menu.html - Menu with food categories
- about.html - About us
- gallery.html - Food gallery
- blog.html - Blog posts
- faq.html - Frequently asked questions
- contact.html - Contact form
- login.html - Sign in page
- signup.html - Sign up page

## Technologies
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Features
- Fully responsive layout on mobile, tablet and desktop
- Bootstrap navbar with mobile collapse
- Shopping cart with add to cart popup on menu page
- Contact form with JavaScript validation
- Food gallery with images
- FAQ accordion
- Sign in and sign up pages

## How to run
Open index.html in a browser. No build tools required.
